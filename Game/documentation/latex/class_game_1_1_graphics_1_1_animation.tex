Metoda de update care se ocupa cu acualizarea index-\/ului frame-\/ului ce trebuie randat. 

timer este incrementat cu o cantitate de timp egala cu diferenta dintre momentul actual de timp si momementul in care a fost randat prima data frame-\/ul actual

last\+Time este actualizat la momentul actual de timp

Daca cantitatea de timp pentru care a fost randat frame-\/ul actual este mai mare decat timpul de asteptare pentru randarea urmatorului frame , indexul este incrementat, iar timer este resetat pe 0

Daca index-\/ul este mai mare de cat numarul de frame-\/uri pentru animatie, atunci acesta este resetat pe 0 astfel, animatia o va lua de la inceput
