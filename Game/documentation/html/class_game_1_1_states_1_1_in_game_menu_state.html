Detailed Description

Clasa se comporta similar cu cea de MenuState, doar ca sunt adaugate functionalitati ce ii permite user-ului sa isi salveze progresul in baza de date, iar daca doreste, cand redeschide jocul sa poate continua cu datele salvate.
