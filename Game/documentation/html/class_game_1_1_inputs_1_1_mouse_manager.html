◆ mouseReleased()

public void Game.Inputs.MouseManager.mouseReleased ( MouseEvent  mouseEvent)
inline

Metoda suprascrisa ce are rolul de a reseta flagurile leftPressed si rightPressed, dar si de a apela metoda onMouseRelease din managerul User Interface.

Daca nu mai este apasat click stanga, atunci se reseteaza flagul leftPressed

Altfel, daca nu mai este apasat click dreapta, atunci se reseteaza flagul rightPressed

Daca avem un uiManager de la care primim input, atunci se apeleaza onMouseRelease, astfel se va parcurge prin toate obiectele grafice si se va apela metoda onMouseRelease, care la randul sau va apela metoda onClick care va realiza task-ul implementat pentru obiectul respectiv
